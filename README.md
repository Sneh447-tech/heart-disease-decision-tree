# Heart Disease Prediction using Decision Tree

This project builds a machine learning model to predict heart disease using the Heart dataset.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model Used
Decision Tree Classifier

## Techniques Used
- Train Test Split
- RandomizedSearchCV
- GridSearchCV
- Hyperparameter Tuning
- Cross Validation

## Steps
1. Load dataset
2. Split data into training and testing
3. Train Decision Tree model
4. Tune hyperparameters using Random Search
5. Tune hyperparameters using Grid Search
6. Evaluate accuracy

## Results
The model achieved high accuracy after hyperparameter tuning.

## Dataset
Heart Disease Dataset
